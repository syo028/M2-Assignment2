DAE 手機應用程式 API 伺服器

歡迎使用 DAE 手機應用程式 API 伺服器。

詳細資料請參閱 assignment-2.md 檔案。

所有資料均由人工智能生成，不保證資料的準確性。

此 API 端點僅供教育用途。

API 端點

資源端點

  • GET /api/courses - 程式教學
  • GET /api/exercises - 運動教學
  • GET /api/yoga-poses - 瑜伽動作
  • GET /api/software - 開源軟體
  • GET /api/hardware - 開源硬體
  • GET /api/fonts - 免費字體
  • GET /api/classical-music - 古典音樂
  • GET /api/wiki-entries - 維基百科
  • GET /api/attractions - 公眾景點
  • GET /api/pet-breeds - 寵物品種

使用者認證

  • POST /api/auth/signup - 註冊
  • POST /api/auth/login - 登入
  • GET /api/auth/check - 檢查是否已登入

收藏功能

  • GET /api/bookmarks - 取得收藏清單
  • POST /api/bookmarks/{item_id} - 加入收藏
  • DELETE /api/bookmarks/{item_id} - 移除收藏
